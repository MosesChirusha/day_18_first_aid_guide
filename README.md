# First Aid Guide (Day 18)

Offline-first, browser-based first aid reference designed to provide quick,
clear emergency instructions when access to medical services or internet
connectivity is limited.

Built as part of a 90-day challenge to create practical digital solutions for
healthcare, education, and public access.

---

## Problem
Limited emergency care knowledge can lead to delayed or incorrect responses
during critical situations. In many communities, professional medical help
may not be immediately available, making basic first aid awareness essential.

---

## Solution
An interactive illustrated first aid manual that runs entirely in the browser
and works offline. The guide organizes common emergencies into simple visual
cards and provides step-by-step instructions designed to be easy to follow
under stress.
